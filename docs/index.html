mystarhistory

A self-hosted, hand-drawn star-history chart generator for GitHub repos. Born as a response to the 2026 GitHub stargazers API restriction, inspired by star-history.com. Python CLI, Docker GitHub Action, SVG committed to your repo on an orphan branch.

See the repository for setup, usage, and current limitations (PAT required since Jul 14 2026).
